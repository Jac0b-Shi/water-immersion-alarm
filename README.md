# 浸水警报器 - 上大智造桑多涅

#### 介绍
浸水警报器 - 基于STM32F103/CH32V203的浸水警报器，开发中...

#### 软件架构

浸水警报器软件采用模块化设计，计划主要包括以下几个核心模块：

##### 1. 传感器数据采集模块
- 水位检测：通过防水传感器检测是否发生浸水情况
- ADC驱动：用于读取模拟传感器数据

##### 2. 报警控制模块
- 报警触发逻辑：根据传感器数据判断是否需要报警
- 多级报警机制：支持不同水位程度的不同报警方式

##### 3. 通信模块
- 无线通信：支持WiFi/NB-IOT无线通信方式
- 协议栈：实现MQTT/HTTP等物联网协议
- 远程配置：支持通过云端对设备进行远程配置

##### 4. 用户交互模块
- LED指示：通过不同颜色LED显示设备状态

##### 5. 系统管理模块
- 低功耗管理：在待机状态下降低系统功耗
- 看门狗监控：确保系统稳定运行
- 数据存储：EEPROM存储配置参数和历史记录

##### 6. 电源管理模块
- 电池电量检测
- 低电量提醒

系统采用实时循环架构，主循环中定期检查各模块状态并作出相应处理。关键事件通过中断方式进行处理，确保及时响应紧急情况。
